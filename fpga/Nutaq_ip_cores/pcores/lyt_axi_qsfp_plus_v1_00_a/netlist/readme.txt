-------------------------------
Coregen files
-------------------------------

In this core, the following source files are generated by Coregen:

- frame_check.vhd (used as is)
- frame_gen.vhd (used as is)
- v6_gtxwizard_v1_12.vhd (modified to include REF_CLK_FREQ generic)
- v6_gtxwizard_v1_12_gtx.vhd (modified to include REF_CLK_FREQ generic)

To regenerate the files, open qsfp_plus_cores.cgp in Coregen and
generate the v6_gtxwizard_v1_12 core. The frame_gen and frame_check
files will be in the example design folder.
